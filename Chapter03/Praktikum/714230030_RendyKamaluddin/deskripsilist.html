Istilah istilah dalam internet

Web
Web, berarti jaringan yang dapat menampilkan data-data dan informasi lain melalui Internet.
world wide web
WWW merupakan kumpulan penyedia web untuk dipakai secara massal di seluruh dunia. Di dalam WWW, semua dokumen, menu, indeks, dan lainlain ditampilkan kepada para pengguna Internet sebagai objek dalam format HTML yang dapat dilihat dengan menggunakan Web Browser.
Web Site
Web site merupakan sebuah alamat di WWW yang menyediakan informasi tertentu.
